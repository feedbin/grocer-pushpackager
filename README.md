Builds a PushPackage for Apple's APN for use with Safari/Mavericks
## Installation

Add this line to your application's Gemfile:

    gem 'grocer-pushpackager'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install grocer-pushpackager
